Configurar la base de datos compartida para que todo el equipo pueda ver y editar las iniciativas en tiempo real.

Pasos para conectar (1 sola vez)

  1. Crear un proyecto gratis en supabase.com
  2. Ir a SQL Editor y pegar el SQL de abajo
  3. Ir a Settings > API y copiar la URL y la anon key
  4. Pegarlos acá abajo y listo
Ver SQL para crear la tabla (copiar y pegar en Supabase SQL Editor) 
CREATE TABLE initiatives (
  id BIGSERIAL PRIMARY KEY,
  squad TEXT,
  competidor TEXT DEFAULT '',
  tarea TEXT NOT NULL,
  dolor TEXT DEFAULT '',
  detalle TEXT DEFAULT '',
  tipo TEXT DEFAULT 'QW',
  prioridad TEXT DEFAULT 'Media',
  equipo TEXT DEFAULT '',
  impacto TEXT DEFAULT 'Medio',
  esfuerzo TEXT DEFAULT 'Medio',
  kpi TEXT DEFAULT '',
  estado TEXT DEFAULT 'Backlog',
  fecha TEXT DEFAULT '',
  asignado TEXT DEFAULT '',
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

ALTER TABLE initiatives ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Allow full access" ON initiatives
  FOR ALL USING (true) WITH CHECK (true);

ALTER TABLE initiatives REPLICA IDENTITY FULL;

-- Tabla para materiales/recursos

CREATE TABLE subtasks (
  id BIGSERIAL PRIMARY KEY,

  owner TEXT DEFAULT '',

);

▼

Merchant Sellers (Competitors)

Tier 1 · Migración

Migrar es un riesgo: puedo perder datos, tiempo y ventas

Racional — El costo percibido de migrar (miedo al downtime y pérdida de datos) es el principal detractor.

“Migrar no es empezar de cero. Automágico importa tu catálogo completo vía API. LitExtension migra productos, clientes e historial desde cualquier plataforma. Podés operar en paralelo mientras migrás: cero downtime. CeCe Piume migró de Empretienda en un día y no perdió ni una venta.”

Tier 1 · Techo

Llegué al techo: mi plataforma ya no me alcanza

Racional — Falta de herramientas avanzadas de marketing, upsell y automatización en plataformas de entrada.

“Tu negocio creció, tu plataforma no. Tiendanube tiene +200 apps, 65+ diseños, automatizaciones de carrito abandonado y Marketing Nube ($169M USD en ventas asistidas). Strawberry Style migró de Empretienda y escaló su operación completa. Cuando tu negocio pide más, Tiendanube responde.”

Tier 2 · Costos

Los costos me comen el margen y no son predecibles

Racional — Los costos no percibidos y las comisiones altas impactan la rentabilidad del merchant.

“WooCommerce es ‘gratis’ hasta que sumás hosting + plugins + desarrollo + parches. En Tiendanube pagás en pesos, sabés exactamente cuánto te cuesta, y con Pago Nube ahorrás 21.6% en procesamiento. Sin sorpresas.”

Tier 2 · Soporte

Cuando tengo un problema, nadie me entiende

Racional — Necesidad de un socio que entienda la complejidad logística, fiscal y de pagos en Argentina.

Tiendanube tiene soporte humano 24/7 en tu idioma, que entiende tu operación. Pago Nube, Envío Nube y Marketing Nube no están adaptados para Argentina: son nativos. Cuando algo falla, lo resolvemos juntos.

Tier 3 · Tráfico

Tengo la tienda pero no logro vender: no genero tráfico ni convierto

Racional — Manejan la operación, pero les falta conocimiento en estrategias de marketing para generar tráfico propio.

“TN no es solo tu tienda — es tu escuela de negocios. Calculá tus costos, aprendé a hacer marketing, y accedé a expertos que ya pasaron por donde estás.”

Social Sellers

Tier 1 · Instagram

Una página? A mí no me sirve, yo vendo por Instagram

Racional — La tienda se percibe como costo; CVR muy bajo en “ya vendo en redes”. Muchos ven Tiendanube solo para negocios más grandes.

“Tiendanube no es una página web — es la infraestructura detrás de tu negocio en redes. Es tu catálogo, tu cobro y tu envío, todo integrado a tus redes. Experiencia Sabia empezó igual que vos. Vendé más por redes sociales con menos esfuerzo.”

Tier 2 · Tiempo

Es difícil / No tengo tiempo

Racional — No puede parar WA/IG; setup percibido largo; poco retorno al admin. No se percibe el valor real de tener una tienda en términos de ahorro de tiempo con automatizaciones y procesos ya estandarizados.

“No necesitás saber de tecnología. Miles de emprendedores como vos ya lo hicieron — y si necesitás una mano, te ayudamos a armarlo.”

Tier 2 · WhatsApp

Yo cierro todo por WhatsApp, la tienda sola no me vende

Racional — Venta consultiva; el checkout solo no replica el cierre.

“Tu tienda no reemplaza a WhatsApp — lo potencia. Catálogo profesional + link de pago + envío resuelto. Vos seguí cerrando por WA, nosotros nos encargamos del resto.”

Tier 3 · Formalizar

Me da miedo formalizar

Racional — ARCA/AFIP como barrera emocional; abandono antes de pagos.

“Empezá gratis, sin compromiso. Cuando estés lista para formalizar, te acompañamos con guías y herramientas — a tu ritmo.”

Tier 3 · Registro

Me registré, entré y no entendí por dónde arrancar

Racional — Panel pensado para quien ya conoce e-commerce; caída fuerte post D1.

“No estás solo. Si te trabaste, te ayudamos a terminar de armar tu tienda. Y si ya la armaste — mirá cómo quedó.”

Tier 3 · Tráfico

Armé la tienda y no entra nadie

Racional — El tráfico no llega solo; hay que conectar audiencia de redes con la tienda.

“Ya tenés la audiencia — te falta conectarla. Con un link en tu bio y Marketing Nube, tus seguidores se convierten en clientes.”

Transversal · Aprendizaje

Aprendo a los ponchazos, nadie me enseñó

Racional — Falta de conocimiento de negocio y de las herramientas del ecosistema; TN tiene recursos y educación que no llegan al perfil SS.

“TN no es solo tu tienda — es tu escuela de negocios. Calculá tus costos, aprendé a hacer marketing, y accedé a expertos que ya pasaron por donde estás.”
